# AI大模型舆情监控系统

这是一个专门针对AI大模型回答质量与舆情信源监控的控制台应用。

## 模块介绍
- **项目管理**：支持创建、管理多个监控项目，配置需要监控的AI大模型池及舆情关键词/问题锚点。
- **实时舆情**：按照时效生成舆情新闻动态流，监控舆情级别、扩散趋势与扩散路径，结合顶部通知中心实现实时告警。
- **关键词与问题分析**：(新增)融合传统关键词舆情与AI问题舆情，支持细颗粒度下钻、意图问题簇分析及多维交叉洞察关联。
- **舆情总览**：展示监控模型总数、样本量、平均错误率、高风险预警等核心看板信息。
- **模型诊断**：提供Doubao、Deepseek、Hunyuan、Qwen等模型详细的错误分类统计与数据下钻。
- **信源监控**：监控全网信源分布、识别高风险来源，并提供GEO优化策略的落地指导。
